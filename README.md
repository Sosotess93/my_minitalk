# my_minitalk
Project that contains a server and a client which communicate between us only with signal in binary mod.

# Installation && Tools
Just need to run 'make' on linux system.

Usage
./server1 -> wait an PID in output
./client1 [PID_SERVER] [MSG_TO_SEND]
